Compute Kernel density estimates (KDE) of raster-domain salinity distributions.

Rationale:
- Reads raw salinity pixel values generated by the S5
- Compute KDE curves for months with scipy.stats.gaussian_kde (bandwidth selection as SciPy default: Scotts rule)
- Export the KDE values on a common grid
- Writes the effective KDE bandwidth table

Inputs:
- CSV with raw salinity valures extracted from each raster grid cell

Requirements:
numpy
pandas
matplotlib
scipy
